常见问题解答

不需要，我们的音频测试工具直接在您的网络浏览器中运行。您不需要下载或安装任何软件。您只需要使用现代浏览器，如Chrome、Firefox、Edge或Safari。

对于麦克风测试和一些高级扬声器测试（如延迟测试），我们的工具需要访问您的麦克风来分析音频输入。我们不会记录或存储任何音频 - 所有处理都在您的浏览器本地进行。

是的，我们的工具设计为可以在移动设备和桌面电脑上使用。界面会根据您的屏幕大小进行调整。由于Web Audio API实现的差异，一些高级测试功能在桌面浏览器上可能效果更好。

频率扫描测试会播放一个音调，从低频（20Hz）逐渐变化到高频（20kHz）。这有助于您检查扬声器或耳机是否能够重现人类听力的全部范围，并识别音频系统在某些频率上可能存在的问题。

我们的测试提供对您的音频设备的良好一般评估。然而，对于专业音频工作，我们建议使用专业校准设备。测试准确性取决于您的浏览器、硬件和环境。结果应该被视为信息性而非确定性测量。

麦克风故障排除

未检测到麦克风

  1. 检查麦克风是否正确连接到您的设备
  2. 确保您已允许浏览器访问麦克风
  3. 如果有多个设备，尝试选择不同的麦克风
  4. 重启浏览器并再次尝试
  5. 检查系统设置中麦克风是否被禁用

麦克风不录制声音

  1. 检查系统设置中麦克风是否静音
  2. 增加系统声音设置中的输入音量
  3. 确保没有其他应用程序正在使用麦克风
  4. 尝试说话更大声或靠近麦克风
  5. 如果可能，用其他麦克风测试

音频质量差

  1. 检查环境中的背景噪音
  2. 确保麦克风没有被覆盖或阻挡
  3. 尝试调整您与麦克风之间的距离
  4. 检查麦克风是否需要清洁
  5. 更新您的音频驱动程序

回音或反馈问题

  1. 降低扬声器音量
  2. 测试期间使用耳机代替扬声器
  3. 将麦克风远离扬声器
  4. 检查系统中是否启用了回音消除
  5. 如果问题持续存在，考虑为您的房间进行声学处理

扬声器故障排除

扬声器没有声音

  1. 检查扬声器是否已通电并正确连接
  2. 确保系统音量未静音或音量过低
  3. 验证系统中是否选择了正确的音频输出设备
  4. 尝试从不同的应用程序播放音频
  5. 重启浏览器和计算机

声音失真

  1. 降低音量看是否停止失真
  2. 检查连接线是否损坏
  3. 尝试不同的音频源以排除内容问题
  4. 更新音频驱动程序
  5. 检查扬声器是否需要维修或更换

音频只在一个声道播放

  1. 检查系统的平衡设置
  2. 检查所有连接线
  3. 尝试播放不同内容以确保不是源材料的问题
  4. 用不同的扬声器或耳机测试
  5. 更新音频驱动程序

音频延迟或延迟

  1. 关闭可能使用音频的其他应用程序
  2. 检查系统更新
  3. 更新音频驱动程序
  4. 尝试使用不同的浏览器
  5. 对于蓝牙扬声器，尝试将其移近您的设备
